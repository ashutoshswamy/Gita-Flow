# Gita Flow

Seek wisdom and gentle guidance from the Bhagavad Gita, powered by AI. Gita Flow provides personalized spiritual guidance by channeling the timeless teachings of the Bhagavad Gita through modern AI technology.

## 🌟 Features

- **AI-Powered Wisdom**: Get thoughtful responses inspired by the Bhagavad Gita's teachings
- **Natural Conversation**: Interact with an AI that embodies Krishna's compassionate guidance
- **Responsive Design**: Beautiful, accessible interface that works on all devices
